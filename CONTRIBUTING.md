# Contributing to JN Automation

Vielen Dank, dass Sie zu JN Automation beitragen möchten!

## Code of Conduct

Bitte befolgen Sie unseren [Code of Conduct](./CODE_OF_CONDUCT.md) in allen Interaktionen.

## Wie Sie beitragen können

### Bugs melden

Bugs sollten als GitHub Issues gemeldet werden:

- **Verwendung eines aussagekräftigen Titels**
- **Beschreibung des genauen Problems**
- **Schritte zur Reproduktion**
- **Beobachtetes vs. erwartetes Verhalten**

### Features vorschlagen

- **Erstellen Sie ein GitHub Issue**
- **Erklären Sie warum es nützlich wäre**
- **Führen Sie Beispiele an**

### Code beitragen

1. Fork das Repository
2. Erstellen Sie einen Branch (`git checkout -b feature/amazing-feature`)
3. Committen Sie Ihre Änderungen (`git commit -m 'feat: add amazing feature'`)
4. Push zum Branch (`git push origin feature/amazing-feature`)
5. Öffnen Sie einen Pull Request
